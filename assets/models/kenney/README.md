# Kenney assets used by Drone Simulator

These assets are distributed by Kenney under the Creative Commons CC0 license.

- Nature Kit: forest ground, paths, water, trees, rocks, stumps and logs.
- Survival Kit 2.0: base camp, rescue platform, supplies and campfire.
- Factory Kit 3.0: charging station equipment.
- 3D Road Tiles: retained as a sample for a future urban scene; Mission 2 uses
  Nature Kit paths to preserve its forest visual language.

Only the models used by the simulator are copied into this directory. Keep the
`Textures` folders beside the Survival and Factory GLB files because those GLBs
reference `Textures/colormap.png`.
