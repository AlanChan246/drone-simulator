請用本機網頁伺服器開啟專案

目前是以 file:// 開啟頁面，瀏覽器會擋下 3D 模型載入（Cross-Origin / 本地檔案安全限制），因此無人機與建築 GLB 都無法讀取。這不是程式 bug，請改在專案根目錄啟動 HTTP 後再開網址。

步驟：在終端機進入本專案資料夾後執行其一：

python3 -m http.server 8080

於瀏覽器開啟 http://localhost:8080（或終端機顯示的埠號）。

Windows 若沒有 python3，可試 py -m http.server 8080；或使用 VS Code 的 Live Server 擴充功能。專案亦提供 npm start（見 package.json）。
